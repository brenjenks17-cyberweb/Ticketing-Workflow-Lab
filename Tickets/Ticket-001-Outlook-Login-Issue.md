# Ticket 001 - Outlook Login Issue

## Problem

A user reported that Outlook kept asking for their password and would not connect to their email.

## What I Checked

I asked when the issue started and found out the user had recently changed their password.

I confirmed the account was active and there were no reported outages.

## Troubleshooting Steps

- Checked for saved credentials
- Removed old credentials
- Restarted Outlook
- Had the user sign in again

## Resolution

After removing the old credentials and signing back in, Outlook connected successfully.

The user confirmed they could send and receive email.

## What I Learned

Password changes can sometimes leave old credentials stored on a device, which can cause login issues.

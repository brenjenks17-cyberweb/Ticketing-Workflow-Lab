# Ticket 003 - VPN Connection Issue

## Problem

A user said they couldn't connect to the VPN from home. Every time they tried to connect, it would fail after a few seconds.

## What I Checked

First, I made sure their internet was working since the VPN won't connect without it. I also confirmed they were using the correct username and password.

## Troubleshooting Steps

* Confirmed the user had internet access
* Verified login credentials
* Reviewed VPN settings
* Removed the existing VPN profile
* Set up a new VPN profile
* Tested the connection

## Resolution

After setting up a new VPN profile, the connection worked normally. The user was able to access company resources again.

## What I Learned

At first I thought it was a password issue, but the problem ended up being the VPN profile itself. Recreating it fixed the issue.

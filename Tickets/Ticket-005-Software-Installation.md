# Ticket 005 - Software Installation Request

## Problem

A user needed software installed on their computer for a project they were working on.

## What I Checked

Before installing anything, I confirmed the software had been approved and would run on the user's system.

## Troubleshooting Steps

* Verified the request was approved
* Downloaded the installer
* Completed the installation
* Opened the application
* Confirmed it launched correctly

## Resolution

The software installed successfully and the user confirmed everything was working as expected.

## What I Learned

Even simple requests like software installations should be documented. Taking a few extra minutes to verify approval and test the application can prevent issues later.

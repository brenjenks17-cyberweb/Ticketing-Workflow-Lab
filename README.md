# Ticketing-Workflow-Lab

I created this project to practice documenting and resolving common IT support issues.

The goal is to show how a help desk ticket moves from the initial report through troubleshooting and resolution.

## What I Practiced

- Documenting user issues
- Troubleshooting problems
- Recording steps taken
- Writing clear resolutions
- Closing tickets

## Tickets Included

- Outlook login issue
- Printer offline
- VPN connection problem
- Account lockout
- Software installation request
